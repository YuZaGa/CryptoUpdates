# CryptoCurrencyNewsNetwork
A cryptocurrency django project is a project that uses cryptocompare api and display latest news about crypto currency and shows the currency data.

<h2>Prerequisites</h2>
<code>See the requirements.txt file</code>

<h2>Installing</h2>
<pre>open terminal and type</pre>
<code>git clone https://github.com/devmahmud/CryptoCurrencyNewsNetwork.git</code><br><br>

<h4>or simply download using the url below</h4>
<code>https://github.com/devmahmud/CryptoCurrencyNewsNetwork.git</code><br>

<h2>To migrate the database open terminal in project directory and type</h2>
<code>python manage.py makemigrations</code><br>
<code>python manage.py migrate</code>

<h2> To run the program in local server use the following command </h2>
<code>python manage.py runserver</code>

<p>Then go to http://127.0.0.1:8000/home in your browser</p>
